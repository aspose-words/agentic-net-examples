# AGENTS.md - Range

## Purpose

This folder is a **live, curated example set** for Range-based operations including search, replace, extraction, bookmarks, and fields.

## Conventions

- Use documented Aspose.Words APIs only.
- Create local sample inputs when needed.
- Keep validation narrow and task-specific.
- Save outputs when required.
